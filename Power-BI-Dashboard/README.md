# Power BI Dashboard

Place Power BI files and dashboard-related assets here.

## Suggested Files

- `.pbix` Power BI report file
- exported PDF version of the dashboard
- dashboard theme JSON, if used

## Suggested Dashboard KPIs

- Total loan applications
- Total loan amount
- Approval rate
- Default rate
- Average credit score
- Average interest rate

## Suggested Visuals

- Default rate by credit score band
- Loan amount by loan purpose
- Default rate by employment status
- Approval status distribution
- Income versus loan amount scatter plot

## Dashboard Design Guide

Use `dashboard_design_guide.md` for a professional Power BI layout plan, KPI cards, chart recommendations, DAX measures, slicers, and formatting tips.
