# SQL

Store SQL scripts used for data cleaning, exploration, and analysis.

## Suggested Files

- `loan_risk_analysis_queries.sql` - main analysis queries
- `data_cleaning.sql` - optional cleaning script
- `kpi_queries.sql` - optional dashboard KPI queries

## Beginner Workflow

1. Import the dataset into a SQL database.
2. Inspect columns and data types.
3. Check for missing values and duplicates.
4. Create KPIs such as approval rate and default rate.
5. Analyze risk by customer segment.

## Main Analysis Queries

The main file `loan_risk_analysis_queries.sql` includes queries for:

- total applicants
- approved vs rejected loans
- average applicant income
- high loan amount applicants
- loan approval by education
- credit history impact
- risky customer categories
- gender-wise loan analysis
- property area analysis
- dependents vs approval rate
