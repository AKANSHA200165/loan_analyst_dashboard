# Resume Project Description

## Banking Loan Risk Analysis

**Banking Loan Risk Analysis | SQL, Power BI, Excel, Data Cleaning, Dashboarding**

- Analyzed banking loan application data to identify key factors affecting loan approval, rejection, and customer risk.
- Cleaned and prepared loan data using Excel and structured data cleaning techniques, including handling missing values, duplicates, inconsistent categories, and standardized column names.
- Wrote SQL queries to calculate total applicants, approval rate, rejection rate, average income, average loan amount, credit history impact, and segment-level loan performance.
- Built an interactive Power BI dashboard with KPI cards, slicers, and visualizations for loan status by education, gender, property area, credit history, and dependents.
- Generated business insights to identify high-risk customer segments based on credit history, income patterns, loan amount, and repayment capacity.
- Recommended risk-based lending strategies to improve approval decisions, reduce default risk, and support better loan portfolio management.
